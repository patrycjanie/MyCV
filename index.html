Kursy

Udemy.com - Web Developer Bootcamp by Colt Steele

Kurs obejmujący podstawy wszystkich technolgii potrzebnych do stworzenia strony internetowej, od HTML i CSS, poprzez JavaScript, aż do Node.js. oraz MongoDB.

Udemy.com - CSS3 for beginners

Kurs obejmujący podstawy języka CSS wraz z krótkim omówiniem poszczególnych funkcji.

Udemy.com - Advanced CSS and Sass: Flexbox, Grid, Animations and More!

Obszerny kurs obejmujący swoim zakresem zaawansowane funkcje CSS3, przy pomocy których tworzyć można nowoczesne strony internetowe. Pozwala poznać narzędzia i sposoby na tworzenie animacji, pozycjonowanie elementów za pomocą css grid oraz flexbox. Pomaga w nauce pisania stron internetowych w Sass. Kładzie nacisk na budowanie responsywnych stron przy pomocy komponentów oraz na pisaniu kody reużywalnego.

freeCodeCamp.org

Kurs pomagający w pozyskaniu umiejętności takich jak HTML5, CSS3, Sass, Bootstrap, JavaScript, jQuery, Git&Github, Node.js i wiele innych poprzez rozwiązywanie zadań.

MDN web docs - Getting started with the Web

Obszerna i szczegółowa dokumentacja pozwalająca na dokładne zapoznanie się z poszczególnymi etapami powstawania strony internetowej.
